1. Two Sum(Easy)

2018-12-01
  • 算法

题目描述

原题

Description:

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Example:

Given nums = [2, 7, 11, 15], target = 9,

Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].

原题翻译

描述：

给定一个整数数组，返回两个数字的索引，使它们相加到特定目标。
假设每个输入仅有一个解决方案，且不能两次使用相同的元素。

例如：
给定 nums = [2, 7, 11, 15]， target = 9。
由于 nums[0] + nums[1] = 2 + 7 = 9，
返回 [0, 1]。

解法二

大佬的答案…渐渐自闭…

主要思想

  1. 创建一个 HashMap<Integer, Integer> 类型的map，key 存放数值，value 存放下标。

  2. 循环遍历数组nums，并把key-value放入map，判断在 map 中是否存在 target-nums[i]：

    （1）若存在，直接将两个下标存入 result 数组，并返回；

    （2）若不存在，将 nums[i] 的值和下标放入 map，进入下一次循环。

时间复杂度：$O(n)$
